Waseshall算法

$Waseshall$算法用来求解传递闭包 问题。
一个$n$顶点有向图的传递闭包可以定义为一个$n$阶布尔矩阵$T=${ $ t[i, j] $ }，如果从第$i$个顶点到第$j$个顶点之间存在一条有效的有向路径（即长度大于$0$的有向路径），矩阵第$i$行$（1 \le i \le n ）$第$j$行$（1 \le j \le n）$的元素为$1$，否者，$ t[i, j] $为$0$。
